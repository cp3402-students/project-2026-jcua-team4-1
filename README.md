## Purpose of the Repository
This repository is used to manage and track the development of our CP3402 group project. It contains all project-related files, including the custom WordPress theme and supporting documentation. GitHub is used for version control, allowing all team members to collaborate, make updates, and maintain a clear history of contributions throughout the project. It also ensures that the project is organised, accessible, and properly documented.

## Documentation Links
The following documentation files provide detailed information about different aspects of the project:

- **deployment.md** – Explains the development workflow, including how the project moves from local development to staging and production environments. It also outlines the tools and steps used for deployment.

- **theme.md** – Describes how the custom WordPress theme is built, including its structure, important files, and development approach.

- **site.md** – Provides information about the website content, structure, and how pages and content are organised and managed within WordPress.
